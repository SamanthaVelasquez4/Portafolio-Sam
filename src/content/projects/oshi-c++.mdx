El juego "Oshi" (también conocido como "Push" o "Oshi Sumo") es un juego de mesa abstracto y estratégico originario de Japón, 
inspirado en el sumo, se basa en empujar fichas del adversario fuera del tablero usando mecánicas de jerarquía y movimiento táctico.  


## Experiencia adquirida: 

Durante el desarrollo del juego OSHI en C++, consolidé habilidades en programación estructurada y manipulación de matrices 
bidimensionales para crear un tablero interactivo, implementando lógicas de movimiento jerárquico y validación de reglas. 
Aprendí a optimizar la interfaz de consola mediante el uso de la API de Windows para colores y entradas de teclado, mejorando 
la experiencia de usuario. Además, reforcé técnicas de modularización de código, separando claramente la lógica del juego (turnos, 
empujes, puntuación) en funciones especializadas, lo que facilitó el mantenimiento y escalabilidad. Este proyecto me permitió 
abordar desafíos como la gestión de estados del juego y la depuración de algoritmos complejos, afianzando mi capacidad para 
resolver problemas de manera sistemática y creativa.
