Desarrollé un sistema de inventario para una tienda deportiva en Java, utilizando Swing GUI Designer para agilizar la creación de 
interfaces gráficas. La aplicación gestiona stock, compras, ventas, proveedores, empleados y generación de facturas, con un enfoque 
en usabilidad y eficiencia operativa. 


## Experiencia adquirida: 

Este proyecto demostró mi capacidad para desarrollar sistemas empresariales robustos en Java, combinando el poder de Swing GUI Designer 
para interfaces ágiles con una arquitectura backend sólida. Destaco mi habilidad para traducir requisitos comerciales en funcionalidades 
técnicas eficientes, priorizando la experiencia del usuario final.
